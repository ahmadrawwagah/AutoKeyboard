This keyboard was created by Ahmad Rawwagah

You can press keys on your keyboard to make sounds!

You can also upload a file with AhmadMusicCode in it to play it!

Rules of AhmadMusicCode

  • The first line of the code must be the tempo
  • Every subsequent line should contain comma seperated notes with a beat value at the end denoting how long the note(s) should be held
  • The allowed notes are C,C#,D,D#,E,F,F#,G,G#,A,A#,B in octaves 3 to 5. You must notate the selected octave by adding the number to the end of the note
  • Rests are also allowed and are notated by the letter R
  • The possible beat values are 1/32 1/16 1/8 3/16 1/4 3/8 1/2 3/4 1

    • Example Code

    120

    F#4,D4,B3,1/8

    R,1/8

    A4,F#4,1/8

    C#5,A4,1/8
